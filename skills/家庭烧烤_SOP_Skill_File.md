# 家庭烧烤 SOP（Skill File）

> 持续维护版本：根据家庭烧烤经验总结。

# 目标
建立一套标准化烧烤流程（SOP），减少遗漏，提高安全性和体验。

---

# 一、前一天准备

## 家务
- [ ] 前一天晚上运行洗碗机，把所有脏餐具洗干净，第二天直接使用。

## 食材

### 肉串
- [ ] 所有冷冻肉串提前一晚放入冷藏室解冻。
- [ ] 如果是在大华（99 Ranch）购买的羊肉串、牛肉串，提前用盐腌制。

### 自己制作肉串（可选）
- [ ] 将整块肉放入穿串机。
- [ ] 合上盖子并压紧。
- [ ] 将签子穿入。
- [ ] 穿好后再切成肉串。
- [ ] 从机器中取出。

> 如果购买的是现成肉串，则跳过本步骤。

### 馒头
- [ ] 冷冻馒头无需提前解冻。
- [ ] 建议烧烤当天蒸热，或前一天晚上蒸好。

### 蔬菜
- [ ] 蔬菜提前穿串。
- [ ] 茄子可于烧烤前对半切开并穿串。

### 烤茄子蒜蓉酱
原料：
- 蒜蓉
- 小米辣
- 热油
- 生抽
- 鸡精
- 糖

做法：
1. 蒜蓉、小米辣混合。
2. 用热油泼香。
3. 加入生抽。
4. 加入鸡精。
5. 加入少量糖。

## 饮品
- [ ] 所有饮料提前冰镇。
- [ ] 准备充足过滤纯净水并冷藏。

## 工具维护
- [ ] 用钢丝球清洁烧烤架和烤网。

# 三、现场布置

- [ ] 取下烧烤区域正上方的装饰彩灯。
- [ ] 摆放桌椅。
- [ ] 放置推车。
- [ ] 摆放餐具。
- [ ] 摆放饮料和过滤纯净水。
- [ ] 放置两个纸箱垃圾箱。
- [ ] 放置桌面小金属垃圾桶。
- [ ] 打开背景音乐。

---

# 四、烧烤流程

## 生火
- [ ] 使用起炭桶点炭。
- [ ] 使用引火纸和打火器。
- [ ] 必要时使用扇子助燃。
- [ ] 木炭表面大部分发白后开始烧烤。

## 烧烤
- [ ] 生熟夹分开使用。
- [ ] 馒头蒸热后再烤。
- [ ] 茄子烤软后刷蒜蓉酱。
- [ ] 饮料和纯净水保持供应。

---

# 五、收尾

- [ ] 吃完的签子放入桌面小金属垃圾桶。
- [ ] 包装垃圾放入两个纸箱。
- [ ] 清洁餐具。
- [ ] 用钢丝球清洁烧烤架和烤网。
- [ ] 收拾所有设备。

木炭处理：
- [ ] 木炭自然燃尽。
- [ ] 等炭灰完全冷却。
- [ ] 完全冷却后再倒入垃圾箱。

---

# 六、安全事项

## 必须佩戴
- K95口罩
- 护目镜

## 烧烤过程中
- 不要碰翻烧烤炉。
- 确保烧烤架放置平稳。
- 生熟食分开夹取。

## 烧烤结束
- 不要处理仍有余温的木炭或炭灰。

---

# 七、经验总结（Best Practices）

- 大华成品肉串无需自己穿串。
- 自制肉串先穿签后切。
- 馒头蒸过再烤口感更好。
- 饮料提前冰镇。
- 过滤纯净水提前冷藏。
- 背景音乐提升聚会氛围。
- 吃完的签子统一放小金属垃圾桶。
- 包装垃圾放纸箱，方便最后统一处理。
