# Skill: Amazon Subscribe & Save + Amazon Key In-Garage Delivery

## Summary

This document summarizes what was learned about Amazon Subscribe & Save
deliveries and Amazon Key In-Garage Delivery.

## Key Findings

### 1. Subscribe & Save orders are not eligible for Amazon Key In-Garage Delivery

-   Amazon officially states that Subscribe & Save shipments are **not
    eligible** for Amazon Key In-Garage Delivery.
-   This is an Amazon product policy rather than a user-configurable
    setting.
-   There is currently no setting that allows users to permanently
    switch Subscribe & Save deliveries to garage delivery.

### 2. Regular Amazon orders can use Amazon Key

For eligible orders shipped by Amazon Logistics:

-   During checkout, Amazon may offer:
    -   In-Garage Delivery
    -   Amazon Day with In-Garage Delivery
-   Eligibility is determined individually for each order.

### 3. Why this happens

Although many Subscribe & Save packages are ultimately delivered by
Amazon Logistics, Amazon still excludes them from Amazon Key because:

-   Subscribe & Save orders do not have a monthly checkout where
    delivery options are selected.
-   Carrier assignment is not guaranteed when the recurring order is
    created.
-   Amazon currently does not support Amazon Key for the Subscribe &
    Save workflow.

### 4. Customer service

If Subscribe & Save packages are consistently delivered by Amazon
Logistics, it is reasonable to contact Amazon Customer Service and
request:

-   whether an exception is possible (unlikely), and
-   that a feature request be submitted for Amazon Key support on
    Subscribe & Save.

## Delivery Instructions

### Recommended

A good delivery instruction is:

> If the package fits, please place it through the mail slot in the
> door. Thank you!

Using "mail slot" is clearer than "mail opening."

### Not Recommended

Do **not** place your private garage keypad PIN or access code in:

-   Additional delivery instructions
-   Security code / call box field

Reasons:

-   These fields are intended for building or community access.
-   Drivers are generally not authorized to enter a private garage
    without Amazon Key authorization.
-   Sharing a garage access code creates unnecessary security risk.

## Practical Recommendation

For items requiring secure delivery:

-   Use regular Amazon orders with Amazon Key whenever possible.
-   Continue using Subscribe & Save only if the discount is more
    valuable than in-garage delivery.
-   If necessary, submit feedback to Amazon requesting Amazon Key
    support for Subscribe & Save.

## Notes

Delivery instructions are requests rather than guarantees. Drivers may
decline instructions that are unsafe, impractical, or inconsistent with
Amazon delivery policy.
